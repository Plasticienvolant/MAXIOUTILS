Repères rapides

État des liens

Renseigne les liens une fois, puis ouvre tes outils depuis ici.

Mode d’emploi très court

1. Colle les liens de tes apps et du classeur.
2. Clique sur Mémoriser.
3. Ouvre ensuite tes outils depuis ce tableau de bord.
4. Garde ce fichier comme page d’accueil de ton système.

Ce tableau de bord sert à quoi ?

• centraliser les liens
• éviter de chercher les fichiers
• garder une cohérence visuelle
• préparer l’arrivée d’autres apps plus tard

Suite possible

• ajouter des tuiles vers d’autres apps
• afficher le dernier fichier utilisé
• faire une page d’accueil encore plus graphique
• ajouter des raccourcis “cours”, “évaluation”, “suivi”
Les liens et les notes sont mémorisés dans le navigateur de l’appareil.
